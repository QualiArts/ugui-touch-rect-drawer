# Install

dependenciesに追加します。

```manifest.json
{
  "dependencies": {
    ...,
    "ugui-touch-rect-drawer": "https://github.com/QualiArts/ugui-touch-rect-drawer.git?path=Packages/ugui-touch-rect-drawer",
  }
}
```

お手軽に使うには、シーンにDrawerContainer.prefabを配置します。
場合によってはSingletonなどの配下に配置するのもいいでしょう。
